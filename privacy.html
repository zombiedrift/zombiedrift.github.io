1. Privacy Policy

This Privacy Policy describes how ZombieDrift Studio™ processes player data when using the game Hockey Zombie Multiplayer™.

  • Data Synchronization: Player nicknames, team colors, and skin indices are saved locally on your device using the PlayerPrefs system and synchronized in real-time across the network via the PurrNet framework during multiplayer matches.
  • Network Connection: To organize the game session, authorize lobbies, and transmit gameplay events, PurrNet temporarily processes technical data (IP address, session data, network ping).
  • Personal Data: The game does not collect personal identity information, such as real names, email addresses, passwords, or financial information. All lobby authentication details are anonymous.
  • Local Settings: Any customization changes are stored directly on the game client and can be reset by clearing the game cache.
